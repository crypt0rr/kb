## Chsh

## Usage

```plain
chsh [options] [LOGIN]
```

## Flags

```plain
Options:
  -h, --help                    display this help message and exit
  -R, --root CHROOT_DIR         directory to chroot into
  -s, --shell SHELL             new login shell for the user account
```

## Examples

### Change default shell

```plain
$ chsh -s /bin/zsh

Password:
Changing the login shell for <user>
```
