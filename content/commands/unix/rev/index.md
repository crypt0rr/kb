## Rev - reverse lines characterwise

## Usage

```plain
rev [options] [file ...]
```

## Flags

```plain
Reverse lines characterwise.

Options:
 -h, --help     display this help
 -V, --version  display version

For more details see rev(1).
```

## Examples

```plain
$ echo 'example-string' | rev
gnirts-elpmaxe
```
