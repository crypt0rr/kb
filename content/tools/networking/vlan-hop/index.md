## VLAN hop

Commands needed to do a VLAN hop.

## Installation

```plain
sudo apt install vlan
```

## Usage

```plain
sudo modprobe 8021q
sudo vconfig add <ethernet-adapter> <vlan-id>
sudo dhclient <ethernet-adapter>.<vlan-id>

ifconfig eth0.200

sudo vconfig rem <ethernet-adapter>.<vlan-id>
```
