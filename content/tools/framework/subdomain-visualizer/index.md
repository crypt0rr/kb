## Subdomain Visualizer

Script that visualizes subdomains for the domain you want.

### Installation

```plain
git clone https://github.com/crypt0rr/subdomain-visualizer.git
```

### Usage

```plain
chmod +x subdomain-visualizer.sh
./subdomain-visualizer.sh
```
