## cmedb

cmedb is default installed with your installation of [CrackMapExec]({{< ref "../" >}}).

### Usage

```plain
cmedb
```

#### Switch protocols

```plain
proto smb
proto ldap
```

#### Dump captured passwords plaintext

```plain
cmedb (default)(smb) > export creds plaintext plain.txt
[+] creds exported
cmedb (default)(smb) >
```
