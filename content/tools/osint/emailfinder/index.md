## Emailfinder

Search emails through Search Engines.

## Installation

```plain
python3 -m pip install emailfinder
```

## Usage

```plain
emailfinder [-h] -d DOMAIN [-v]
```

## Flags

```plain
optional arguments:
  -h, --help            show this help message and exit
  -d DOMAIN, --domain DOMAIN
                        Domain to search
  -v, --version         Show Emailfinder version
```
