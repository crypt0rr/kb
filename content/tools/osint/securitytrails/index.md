#### Gather subdomains

```plain
curl --request GET \
  --url 'https://api.securitytrails.com/v1/domain/<DOMAIN-HERE>/subdomains?children_only=false' \
  --header 'APIKEY: <KEY-HERE>' \
  --header 'Accept: application/json'
```
