A debugger and .NET assembly editor. You can use it to edit and debug assemblies even if you don't have any source code available.

## Installation

```plain
git clone --recursive https://github.com/0xd4d/dnSpy.git
cd dnSpy
./build.ps1 -NoMsbuild
```
