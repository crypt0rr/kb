## Inxi

Command line system information script for console and IRC.

## Installation

```plain
sudo apt install inxi
```

## Usage

```plain
inxi [OPTIONS]
```

## Flags

You can use these options alone or together, to show or add the item(s) you want to see: A, B, C, D, E, G, I, J, L, M, N, P, R,
