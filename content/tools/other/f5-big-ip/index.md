## F5 BigIP Decoder

Detecting and decoding F5 BIG IP cookies in bash.

## Installation

```plain
git clone https://github.com/drwetter/F5-BIGIP-Decoder.git
```

```plain
chmod +x f5_bigip_decoder.sh
```

## Usage

```plain
./f5_bigip_decoder.sh '<cookie>'
```
