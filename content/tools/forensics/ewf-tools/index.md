## Ewf-Tools

Collection of tools for reading and writing EWF files.

## Installation

```plain
sudo apt install ewf-tools
```

### Tools

```plain
ewfacquire        ewfdebug          ewfinfo           ewfrecover
ewfacquirestream  ewfexport         ewfmount          ewfverify
```

## Examples

```plain
ewfmount image.E01 <folder>
```
