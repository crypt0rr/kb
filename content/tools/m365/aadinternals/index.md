## AADInternals

AADInternals is PowerShell module for administering Azure AD and Office 365

For details, please visit <http://o365blog.com/aadinternals>

### Installation

```plain
# Install the module
Install-Module AADInternals

# Import the module
Import-Module AADInternals
```
